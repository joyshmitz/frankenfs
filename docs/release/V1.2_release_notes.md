# V1.2 Release Notes

The V1.2 release is gated by `scripts/e2e/ffs_v12_program_gate_e2e.sh`.

The gate writes its durable manifest to:

- `artifacts/release_gate/v12/program_gate_manifest.json`
- `artifacts/release_gate/v12/scenarios/scenario_<n>.jsonl`
- `artifacts/release_gate/v12/command_transcript.tsv`

The manifest recommendation is one of:

| Recommendation | Meaning |
|---|---|
| `PROCEED` | All fourteen scenarios passed and no waiver document was active. |
| `PASS-WITH-WAIVERS` | All scenarios passed, but `docs/release/V1.2_test_waivers.md` is non-empty. |
| `NO-PROCEED` | At least one scenario failed or timed out. |

GitHub tag builds matching `v1.2.*` and manual dispatches run `.github/workflows/v12-program-gate.yml`, which invokes the script and uploads `artifacts/release_gate/v12/` for at least 90 days.
